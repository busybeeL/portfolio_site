Jun 2020 - Dec 2021 Connect: North Korea Connect: North Korea is a charity organisationto support North Korean escapees in the UK. They assist North Korean communities realise their vision to heal, grow and live the life that they choose. I built this website for Connect: North Korea as a volunteer web developer. It is built in customised WordPress theme. The structure of this website consists with static HTML and LESS and the animations are added using the Greensock JavaScript library. It is scalable from desktop to mobile with media query.
